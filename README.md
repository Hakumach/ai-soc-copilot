# AI SOC Copilot

AI-powered cybersecurity incident analysis API built with FastAPI and Python.

This project simulates a lightweight SOC (Security Operations Center) assistant capable of analyzing security logs, detecting suspicious activity patterns, assigning severity levels, and generating automated recommendations.

---

# Features

- Analyze cybersecurity log events
- Detect suspicious login activity
- Identify malware-related events
- Detect reconnaissance/port scanning activity
- Assign threat severity levels
- Generate automated remediation recommendations
- Interactive Swagger API documentation

---

# Tech Stack

- Python
- FastAPI
- Uvicorn
- Pydantic
